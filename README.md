🏢 Google Maps Business Scraper

A Python automation tool that extracts business information from Google Maps using Selenium and exports the data into a professionally formatted Excel workbook.

🚀 Features
- Dynamic scrolling for loading business listings
- Business name and contact information extraction
- Duplicate record removal
- Automated Excel generation
- Professional Excel formatting using OpenPyXL

🛠️ Tech Stack
- Python
- Selenium
- Pandas
- OpenPyXL

📦 Installation
pip install selenium pandas openpyxl

▶️ Run
python maps_scraper.py

📊 Output
- Company Name
- Phone Number
- Formatted Excel Report
